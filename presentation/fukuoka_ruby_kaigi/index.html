Problem 1

Problem 1

10未満の自然数のうち, 3 もしくは 5 の倍数になっているものは 3, 5, 6, 9 の4つがあり, これらの合計は 23 になる. 同じようにして, 1000 未満の 3 か 5 の倍数になっている数字の合計を求めよ.

3 + 5 + 6 + 9 = 23

出展：ProjectEuler 問題1

当時のSkypeでの解答

某春◯さんの解答

#!/usr/bin/env ruby
# encoding: UTF-8

total = 0
(1..1000).each do |num|
  if num % 3 == 0 || num % 5 == 0
    total += num
  end
end
p total

PHPerさんの解答

<?php
//3の倍数
$total = 0;
for($n = 1 ; 3 * $n < 1000 ; $n++){
    $total += 3 * $n;
}

//5の倍数
for($n = 1 ; 5 * $n < 1000 ; $n++){
    //3の倍数は除く
    if( $n%3 != 0 )$total += 5 * $n;
}

echo $total;

#!/usr/bin/env ruby
def sum (max, num1, num2)
  return (1..(max - 1)).select {|n|n % num1 == 0 || n % num2 == 0}.inject(&:+)
end

#!/usr/bin/env ruby
# coding : utf-8
require 'prime'
max = 10001
primes = [2, 3, 5]
n=0
loop do
  break if primes.size > 10001
  n+=1
  prime = 6*n+1
  primes << prime if prime.prime?
  prime +=4
  primes << prime if prime.prime?
end
p primes[max-1]

アルゴリズムを極める

max = 10001
def prime?(num,  ary)
  ary.each do |n|
    break if n * n  > num
    return false if num % n == 0
  end
  return true
end
n = 0
prime_ary = [2,  3,  5]
while  prime_ary.count < max
  n += 1
  prime = 6 * n + 1
  prime_ary.push(prime) if prime?(prime,  prime_ary)
  prime += 4
  prime_ary.push(prime) if prime?(prime,  prime_ary)
end
p prime_ary[-1]
