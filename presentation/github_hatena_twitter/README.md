#### 僕 と github

## githubとは？
 * gitとSNSが融合したサービス
 * 世界中にコードを公開した状態で開発を行うことになる
 * (プライベートでもあるけど)
 * オープンソースソフトウェアや各種ライブラリの開発なども行われてる
 * 最近だとブログなどプログラム以外を公開することに用いられている
